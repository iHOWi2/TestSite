  ФАЙЛЫ СЦЕНАРИЯ (всего 5):

  scenario.txt   → название, описание, стартовые настройки

  events.txt     → 20+ событий с нарративом (4 акта истории)
  govactions.txt → 7 правительственных действий (4 кастомных)
  world.txt      → модификации 10 стран под тематику

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  СПОСОБ 1: ЧЕРЕЗ SCENARIO CREATOR (РЕКОМЕНДУЕТСЯ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Шаг 1. Открой Plague Inc: Evolved → Scenario Creator
Шаг 2. Создай новый сценарий, выбери "Вирус", назови "Припять-X"
Шаг 3. Сохрани и закрой Scenario Creator
Шаг 4. Найди папку сценария:

   Windows:
   C:\Users\{ИМЯ}\AppData\Local\Ndemic Creations\Plague Inc. Evolved\Scenario Creator\Припять-X\

   Mac:
   ~/.local/share/Ndemic Creations/Plague Inc. Evolved/Scenario Creator/Припять-X/

Шаг 5. Замени файлы в папке файлами из этого архива
        (disease.txt, events.txt, govactions.txt, world.txt)
        scenario.txt заменяй осторожно — в нём может быть ID сценария

Шаг 6. Снова открой Scenario Creator → Load → выбери Припять-X
Шаг 7. Нажми Test для проверки

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  СПОСОБ 2: ФАЙЛ .PLAGUEINC (ДЛЯ МОБИЛЬНЫХ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Шаг 1. Положи все 5 файлов .txt в одну папку "pripyat_x"
Шаг 2. Выдели ВСЕ файлы внутри папки (не саму папку!)
Шаг 3. Сожми их в ZIP архив → назови pripyat_x.zip
Шаг 4. Переименуй: pripyat_x.zip → pripyat_x.plagueinc
        (Windows спросит подтверждение — нажми "Да")

На Android:
  → Скопируй .plagueinc на телефон
  → Открой через файловый менеджер
  → Plague Inc. должна предложить импорт

На iOS:
  → Отправь файл себе в Telegram / по email
  → Открой вложение → "Поделиться" → Plague Inc.

         Первые смерти. Границы закрываются.

         Пандемия официальна. Первая вакцина убивает пациентов.

АКТ III: Ядерный вариант обсуждается в ООН. Найдена секретная
         лаборатория. Финальная мутация откатывает лечение на 60%.

АКТ IV:  Правительства падают. Последняя надежда из Женевы.
         Миллиард мёртвых. Последняя радиопередача.
